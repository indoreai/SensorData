About

The Tool is used to analyze data fetched from Sensors. A Device consists of multiple sensors, each measure a specific environment attribute. In our case, we have 5 sensor device which sends 1 records per second in the below format.

Sample HRBME680 RBME680 CO2SGP30 TVOCSGP30 H2SGP30 EtanolSGP30
1 7 1183290366 400 0 14300 19029

We have Deviсе which collect data in regular time interval at different locations. The set of data is called Sample. Generally, A Sample collection time is 10 to 20 minutes.

  • The data is Uploaded to the System using CSV.
  • General Graph plots data of each Sample on per second frequency.
  • PCA is the ML algorithm used to reduce dimension.
  • Suppose we want to plot the data per minute then we have to reduce it to two-dimension from 60.
  • We have done the exact same thing using the skill-learn python library and plot the data.
  • We can apply PCA to multiple Samples and used it for clustering and separation
